C.A.R.L.

(Cvsnt Acl Repository Lacky)


about
C.A.R.L is korn shell script that presents a menu driven frontend for cvsnt ACL (access control lists) configuration. developed in order to update a corporate client to cvsnt while providing a sane way for non-technical personnel to configure and maintain cvs permissions.

Click here to download Carl


requirements
cvsnt (client/server)
korn shell
UNIX (tested on Solaris and FreeBSD but Linux/AIX/HP-UX/OSX will work)
Windows? With Cygwin probably, untested for now.
a sense of humor


usage
tar -xf carl-*.tar
cd carl-*
chmod 755 carl
(edit variables in head of script)
./carl
press D to define directory
press d to define group
use only as prescribed
