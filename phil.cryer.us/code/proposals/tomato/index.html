Proposal for accordion-style navigation for Tomato

Tomato is an Open Source replacement firmware for Linksys', and other Broadcom-based, routers. Besides offering more advanced networking features, it has a web gui for administration. The gui is driven by a static left side menu that is permanently expanded, causing it to scroll off the screen. From a usability standpoint this can be improved.

My proposal expands on the AJAX-ing of the administration GUI by presenting the user with an 'accordion' style menu that expands and contracts depending on which section the user requests. Notice that I have combinded some commands under 'Router Commands' and that the menus 'Access Restriction' and 'About' have no child nodes, but are still linkable, and change color when selected. The code is valid XHTML 1.0 Strict, although it has an IE6 issue which will need to be addressed if accepted. Created on 07/2007 by Phil Cryer, utilizing style from solutoire.com and javascript from Mootools. (46KB uncompressed)
