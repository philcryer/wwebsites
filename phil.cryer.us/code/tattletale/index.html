The Project

I've always said that web logins are broken by today's standards, and don't provide enough security. Having SSL encrypt the transactions works around the problem, but doesn't solve it. Project Tattletale aims to allow for secure transactions of username and passwords for authentication, as well as verifying the user against browser revealed system information. All of this without the user needing to do or install anything additional.
