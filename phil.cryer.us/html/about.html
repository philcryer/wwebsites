All about this site, the methods I used to code it and all of its supporting software

 

web design.

Web page template Simplicity two-point-oh, created by Harish Narayanan, thanks!

        

 

standards.

I recoded the template to XHTML 1.0 Transitional and CSS 2.0 W3C Standards.

            

 

scripts.

Ajax Tabs Content script from Dynamic Drive DHTML scripts.

      

 

editor.

This site completely coded with vim, an improved clone of vi, the venerable open source, Unix text editor. What more do you need?

        

 

operating system.

While I've used Linux for over 10 years now, I ran the server with FreeBSD for almost 5. Its stability and ease of configuration were attractive, but I really just wanted to learn something outside of Linux for awhile, and I really enjoyed it. During the server rebuild of 2007, I opted to go with my favorite Linux distro, the almighty Debian. The box is a one I hand built years ago out of a VIA mainboard, OCZ RAM and an AMD Thunderbird CPU (1200Mhz).
